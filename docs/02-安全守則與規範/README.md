這是一份給國立金門大學針對攀樹與延伸相關活動給予的社團安全說明書，透過說明社團與社團宗旨的相關性來了解與學校的相關性，依序說明該社團在各方面對於安全準備與說明，針對場地、樹木、裝備、教練、社員、攀樹安全守則說明各方向安全與規範，最後解說攀樹師、樹藝師與相關行業的介紹，此份說明書將針對各項風險評估、樹藝師進行說明。

攀樹也可與運動與休閒學系、社會工作學系、觀光管理學系具有直接的相關性，本社團的特色除了基本的「技術攀登」、「繩索技術」與「樹藝師」等，也將攀樹運用在運動、休閒、教育與社會工作等。

攀樹、攀岩、工業繩索不管在運動或職業上都屬於高風險活動，運動更歸類於極限運動，尤其有時需要操作危險機具，但透過正確、安全的知識，加上裝備與工具讓活動與工作提供安全的環境，甚至發揮人對於攀爬本能所帶來正面的樂趣與新的感官刺激。

國立金門大學敏捷攀樹社的宗旨、目的與目標，將在社團的成立後為學校帶來正面的影響與發展，除了成為全國第三個擁有攀樹社的大學，未來有機會成為國立金門大學的特色社團。
